# 批次匯入操作手冊（userstate.html）

本手冊說明 userstate.html「新增資產」中的批次匯入功能操作方式與檔案格式規範，請依實作邏輯準備檔案並依步驟操作。

## 入口位置
- 進入資產管理頁（userstate.html）。
- 右上角點擊「新增資產」。
- 在「新增財產 / 物品」視窗中找到「批次匯入」區塊。

## 支援檔案格式
- Excel：`.xlsx`
- CSV：`.csv`

## 匯入檔案準備規則
### 1) Excel（.xlsx）
- 工作表名稱必須為 `財產總表` 或 `物品總表`（可同時存在兩個工作表）。
- 欄位順序需與匯出格式一致（第一列為欄位名稱，資料從第 2 列開始）。
- 若兩個工作表都有資料，系統會一次匯入財產與物品。

### 2) CSV（.csv）
- 僅能匯入單一類型（財產或物品）。
- 匯入前請先在視窗下方「資料類型」區塊選擇「財產」或「物品」。
- 第一列為欄位名稱，資料從第 2 列開始。

## 欄位順序與必填規則
### 財產（財產總表 / 財產 CSV）
| 欄位 | 說明 | 必填 |
| --- | --- | --- |
| A | 財產編號 | 是 |
| B | 財產名稱 | 是 |
| C | 財產別名 | 否 |
| D | 型號/廠牌 | 否 |
| E | 單位 | 否 |
| F | 取得日期 | 否 |
| G | 使用年限 | 否 |
| H | 保管地點 | 是 |
| I | 附屬設備 | 否 |
| J | 保管人（姓名） | 是 |
| K | 使用人（姓名） | 否（空白會自動帶入保管人） |
| L | 財產類別 | 否（實際以匯入時選擇為準） |

### 物品（物品總表 / 物品 CSV）
| 欄位 | 說明 | 必填 |
| --- | --- | --- |
| A | 物品編號 | 是 |
| B | 物品名稱 | 是 |
| C | 產品序號 | 否 |
| D | 型號/廠牌 | 否 |
| E | 取得日期 | 否 |
| F | 使用年限 | 否 |
| G | 單位 | 否 |
| H | 台幣金額 | 否 |
| I | 申購單號 | 否 |
| J | 財產類別 | 否 |
| K | 保管人（姓名） | 是 |
| L | 保管地點 | 是 |

## 操作步驟
1) 點擊「新增資產」開啟視窗。  
2) 在「批次匯入」區塊選擇 `.xlsx` 或 `.csv` 檔案。  
3) 選擇「財產類別」（財產 / 非消耗品）。  
   - 僅在匯入財產資料時生效，會寫入財產總表 L 欄。  
4) 若為 CSV，請在下方「資料類型」選擇財產或物品。  
5) 點擊「開始批次匯入」。  
6) 匯入完成後畫面會顯示成功/失敗筆數與失敗清單。  

## 系統檢核與常見失敗原因
以下情況會被判定為失敗並列在「失敗清單」：
- 必填欄位缺漏（編號、名稱、保管地點、保管人）。
- 保管人/使用人不是名單中的姓名（請填姓名而非 Email）。
- 名單中有重複姓名（系統要求姓名唯一）。
- 編號已存在於系統中。
- 同一匯入檔案內編號重複。
- Excel 沒有 `財產總表` 或 `物品總表` 工作表。

## 匯入結果說明
匯入完成後會顯示：
- 成功筆數：實際新增到表格的筆數。
- 失敗筆數：驗證未通過的筆數。
- 失敗清單欄位：類型、列號、編號、原因、來源（Excel/CSV/工作表名稱）。

## 注意事項
- 財產/物品編號需全系統唯一（跨兩張表）。
- 日期可用 Excel 日期或 `YYYY/MM/DD`，系統會自動轉換。
- 財產名稱包含「電腦」時，系統會自動標記資訊資產與電腦欄位。
